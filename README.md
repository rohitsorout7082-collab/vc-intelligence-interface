# VC Intelligence Interface 

A professional startup discovery tool designed for Venture Capitalists to search, filter, and manage startup deal flow efficiently.

##  Live Demo
**View Project:** [https://vc-intelligence-interface-jesk.vercel.app](https://vc-intelligence-interface-jesk.vercel.app)

##  Key Features
- **Startup Discovery**: Advanced filtering by sector (Fintech, AI Infrastructure, Clean Energy) and real-time search functionality.
- **Save Searches**: Persistent search filters using `localStorage` to save time on recurring venture searches.
- **AI-Ready Enrichment**: Clean UI structure prepared for AI-driven data extraction and keyword analysis.
- **Responsive Design**: Built with Next.js and Tailwind CSS for a seamless experience on all devices.
- **Robust Build**: Implemented React Suspense boundaries for optimized client-side rendering on Vercel.

##  Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **Language**: TypeScript
